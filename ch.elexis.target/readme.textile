h1. Howto update the Elexis target

To be less dependent on a lot of external downloads links, we assemble
first all our external dependencies into a single repository using
ch.elexis.target/prepare-target and then use the generated repository
http://download.elexis.info/elexis/master/p2/target for the IDE and
and maven builds.

h2. Update steps for prepare-target

# Inside the Eclipse IDE add/remove/change the desired features/plugins by editing @ch.elexis.target/prepare-target/elexis-prepare.tpd@.
#  Via set context menu run "Set as Target Rlatform". This will update @ch.elexis.target/prepare-target/elexis-prepare.target@.
# Verify that the build passes by calling inside the  @ch.elexis.target/prepare-target@ directory the command @ant -DBRANCH_SELECTOR=origin/master -file build.xml build.local.target.platform.repo -DtargetFile=elexis-prepare.target@.
# Commit and push the changes done in @ch.elexis.target/prepare-target@
# Go to "https://srv.elexis.info/jenkins/job/Build_Elexis_branch-productive-target/":https://srv.elexis.info/jenkins/job/Build_Elexis_branch-productive-target/ and trigger a build for the given branch
# Wait till the build output appear in "http://download.elexis.info/elexis/master/p2/target":http://download.elexis.info/elexis/master/p2/target
